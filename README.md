## 🎨 THE DESIGN PHILOSOPHY
This portfolio is built on the principles of **Neubrutalism** — a design language that rejects soft gradients and subtle shadows in favor of:
- **High Contrast:** Bold black borders (`#1F1C18`) and vibrant accent colors.
- **Maximalist Typography:** Heavy use of `Bebas Neue` for titles and `Space Grotesk` for body text.
- **Editorial Layout:** A "Cream & Ink" palette (`#F4F0E6` background) that feels like a high-end film magazine.

## ⚡ PERFORMANCE & OPTIMIZATION
- **Smooth Scroll:** Integrated `Lenis` for a non-jittery scrolling experience.
- **Lazy Loading:** All images use `loading="lazy"` to improve initial FCP.
- **Critical CSS:** A custom loader handles the GSAP initialization to prevent FOIT/FOUC.
- **YouTube API:** Optimized iframe loading with `enablejsapi=1` for seamless modal control.
